Files are placed as follows:

/home/pi/config.txt
  (contains information about limits and email-adresses - description will follow)
  
/home(pi/logs/
  (contains log-files - named after week-number - containing temperature, humidity, presusre and time-stamp - for every 20 minutes)
  
/home/pi/vinecellar/
  (contains all scripts - see scripts-description file)
  
After normal setup (ssh/upgrade/language/wifi)
  - enable ssh in raspi-config
  - sudo apt-get update
  - sudo apt-get upgrade
  - setup language in raspi-config and add values to ~/.bashrc
  - setup wifi-connection i raspi-config

Enable i2c in raspi-config 
sudo apt-get install ssmtp
sudo apt-get install mailutils

edit /etc/ssmtp/ssmtp (contains username, password and server-adress for mail) - a samble file will be included shortly  

wire the bme280 sensor as described here: https://www.raspberrypi-spy.co.uk/2016/07/using-bme280-i2c-temperature-pressure-sensor-in-python/

A great many thanks to Matt Hawkins for the wirering description and the bme280.py (which I edited sligtly to be used ad a module)
